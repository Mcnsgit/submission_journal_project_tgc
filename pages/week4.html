JS DOM

What is the Dom

  • Dom = Document Object Model
  • A representation of a webpage that JS can use.
  • Changes that JS makes to the DOM alter the webpage.

Tree like structure

• A tree of objects • Each object represents an element on the page • Allows JavaScript to access and manipulate the different elements

What can we do?

• Selecting Elements, Reading/Changing Elements, Responding to User Events
