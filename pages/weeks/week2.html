Data Types - SQL

Reference: SQL Data Types at W3Schools

  • INT(size): Integer. Whole numbers. Parameter specifies the maximum display width (up to 255).
  • FLOAT(p): A floating-point number. Can hold decimal places.
  • DECIMAL(size, d): Fixed decimal point number. The total number of digits is specified in the 'size' parameter, and the number of digits after the decimal point is specified in the 'd' parameter. Maximum 'size' is 65, and maximum 'd' is 30.
  • CHAR(size): A fixed-length string (can contain letters, numbers, and special characters). The 'size' parameter specifies the column length in characters, up to 255.
  • VARCHAR(size): A variable-length string (can contain letters, numbers, and special characters). The 'size' parameter specifies the maximum string length in characters, up to 65535.
  • DATE: A date. Format: YYYY-MM-DD. Supported range is '1000-01-01' to '9999-12-31'.
  • BOOLEAN: True or False.

CSS Enhancements and Properties

Throughout this week, we delved into CSS by exploring various properties that enhance the visual appeal and layout of web pages. Key CSS properties we covered include:

  • Transform Property: Used for rotating, scaling, skewing, and moving elements. It's essential for creating dynamic, interactive user interfaces.
  • Box Shadow: Adds shadow effects around elements. We learned to control the shadow's blur, spread, color, and position, adding depth and emphasis to elements.
  • Position: Controls the positioning of elements. We explored different values like static, relative, absolute, fixed, and sticky, understanding their impact on layout and design.
  • Background Property: Used for setting background images and colors for elements. We experimented with gradients, multiple backgrounds, and controlling their size and position.
  • Flexbox: A layout model that allows responsive design with flexible and efficient arrangements of elements within a container.
  • Grid Layout: Provides a grid-based layout system, offering a more structured and versatile method for designing complex layouts.
  • Transitions: For creating smooth animations on CSS property changes, enhancing user experience through interactive feedback.
  • Media Queries: Essential for creating responsive designs, allowing CSS to adapt to different screen sizes and orientations.
